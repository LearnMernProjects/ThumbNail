ThumbNail: AI Thumbnail Generator App
This is a full-stack AI Thumbnail Generator application built using the MERN stack (MongoDB, Express, React, Node.js) and powered by the Google Gemini API. It allows users to generate custom thumbnails for their videos by providing a title, prompt, aspect ratio, color scheme, and style.
